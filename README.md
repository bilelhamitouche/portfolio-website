# Personal portfolio website

This is my personal portfolio website where I showcase my skills as a web developer.

## Installation

To install you first need to clone this repo by typing the following into the command line:

```sh
git clone https://github.com/bilelhamitouche/portfolio-website
```

Then you need to go to the repo and install all the necessary packages by running:

```sh
cd portfolio-website
npm install
```

Then you have to start the developement server by running:

```sh
npm run dev
```

Go to http://localhost:5173 to see the website in action.

## Tech stack used

ReactJS
Typescript
Tailwindcss
DaisyUI
